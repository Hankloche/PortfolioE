Mes Projets

M2L

M2L est une application web et mobile pour gérer les équipes et les tournois. Elle permet de créer des tournois, ajouter des équipes, et suivre les résultats en temps réel.
L'application est développée avec Symfony pour le backend et JavaFX avec SceneBuilder pour le frontend, offrant une interface simple et accessible.

GSB

GSB est une application web et mobile, développée avec **Symfony** pour le backend et **Flutter (Dart)** pour le frontend. Elle permet de gérer facilement les frais professionnels : soumettre, approuver et suivre les dépenses.
